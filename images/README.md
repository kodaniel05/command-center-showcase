# Images

This folder is reserved for public-safe visual assets.

## Screenshot Guidelines

- Use demo projects or redacted data.
- Do not show API keys, tokens, environment variables, private paths, client names, personal data, or proprietary implementation details.
- Prefer full-screen screenshots that show the product clearly.
- Keep filenames lowercase and descriptive.
- Match the placeholder paths in the root `README.md`.

Recommended screenshots:

- `screenshots/dashboard-home.png`
- `screenshots/project-card-runtime-controls.png`
- `screenshots/snapshot-management.png`
- `screenshots/activity-feed.png`
- `screenshots/system-overview.png`
- `screenshots/add-project-modal.png`
- `screenshots/project-logs.png`
- `screenshots/project-notes.png`
- `screenshots/idea-blueprint-studio.png`
- `screenshots/blueprint-output.png`
- `screenshots/settings-view.png`

## Diagrams

Architecture diagrams can be stored in `images/diagrams/` if exported from Mermaid or another diagramming tool. Keep the Mermaid source in the documentation when possible so it remains easy to review.
